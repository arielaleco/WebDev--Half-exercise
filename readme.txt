ראשי (דף הבית)
צור קשר
מצורפים צילומי מסך של שלושת הדפים - בקובץ ה-zip: 'תרגיל אמצע - צילומי מסך', בהם תוכלו להעזר כדי לראות איך הדפים צריכים להראות.


מצורפות התמונות בהן עליכם להשתמש במהלך בניית האתר - בקובץ ה zip: 'תרגיל אמצע - תמונות לאתר'.

הגשה
יש להעלות ל GitHub ולהגיש את הקישור לפרויקט.


הקריטריונים לבדיקה
·         ההגשה צריכה לכלול את כל הנושאים עליהם עברנו עד לאותה נקודה.

·         הקוד צריך להיות נקי ולכלול טכניקות שלמדנו עד אז (למשל: הפרדת CSS לקובץ נפרד, שימוש חוזר ב-CSS וכו׳).

·         אסור להעתיק את הקוד מהאתר המקורי.

·         התוצאה הסופית צריכה להיות דומה למקור, אך אני לא מתעקש על דיוק של פיקסלים.

·         במקרה ולא יודעים איך להגיע לתוצאה מסויימת, אפשר ליצור פתרון חלופי (למשל: אם לא יודעים כיצד ליצור איזור של תמונות מתחלפות - אפשר לשים תמונה סטאטית).
